== Data object characteristics
[cols=".^1h,2,2" , options="header"]]
|===
|  | Input | Description

| Data Sensitivity | | _Describe the level of sensitivity of the data (e.g., public, internal, confidential, restricted)._
| Data Retention | | _Specify the duration for which the data should be retained._
| Data Encryption | | _Indicate whether the data needs to be encrypted and the type of encryption required._
| Data Masking | | _Describe any data masking requirements to protect sensitive information._
| Data Anonymization | | _Explain if and how the data should be anonymized to protect individual identities._
| Data Pseudonymization | | _Detail any pseudonymization techniques to be applied to the data._
| Data Deletion | | _Outline the process and conditions for data deletion._
| Data Archiving | | _Specify the archiving requirements for the data._
| Data Backup | | _Describe the backup procedures and frequency._
| Data Recovery | | _Explain the data recovery process in case of data loss._
| Data Validation | | _Detail the validation checks to ensure data accuracy and integrity._
| Data Transformation | | _Describe any transformations the data undergoes during processing._
| Data Quality | | _Specify the quality standards and metrics for the data._
| Data Governance | | _Outline the governance policies and procedures for managing the data._
| Data Lineage | | _Describe the data lineage to track the data's origin and transformations._
| Data Catalog | | _Indicate if the data should be included in a data catalog and any relevant details._
| Data Dictionary | | _Provide a description of the data elements and their definitions._
|===
